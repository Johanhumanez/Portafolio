# 🛒 Customer Segmentation & Retention Analysis (Olist)

## 📊 Project Overview

This project analyzes customer behavior to identify revenue growth opportunities through retention strategies.

The analysis was performed using SQL, Python, and Power BI.

---

## 🧰 Tools Used

* SQL (data extraction)
* Python (Pandas, data analysis)
* Power BI (dashboard & visualization)

---

## 📂 Dataset

Brazilian e-commerce dataset (Olist)

---

## 🔍 Key Insights

* 97% of customers make only one purchase
* Repeat customers generate ~2x more value ($308 vs $160)
* Increasing retention by 1% → +$154K in revenue
